# **App Name**: CurrencyTrack

## Core Features:

- Savings Entry: Allow users to input their savings data, including amount and currency (USD or SLL).
- Currency Conversion: Automatically convert all savings to USD for total sum calculation. Leverage a reliable API to convert SLL to USD.
- Real-time Calculation: Display the total savings sum in USD, updating in real-time as new entries are added.
- Data Storage: Persist user savings data for future access and tracking.
- Visualization: Present user's saving history through simple data visualization components. LLM acts as a tool deciding how to present it. It can create line chart for datewise amount.

## Style Guidelines:

- Primary color: Forest green (#337A4A) for a feeling of growth and financial security.
- Background color: Light beige (#F5F5DC), creating a neutral and calming backdrop.
- Accent color: Muted gold (#B8860B) for highlights and calls to action, symbolizing wealth and value.
- Body and headline font: 'PT Sans', a modern sans-serif with warmth for readability.
- Use simple, consistent icons to represent different savings categories.
- Clean and intuitive layout with a focus on data clarity and user experience.
- Subtle animations for updating totals and currency conversions.
